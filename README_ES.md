## 📌 Proyectos Destacados

### [Wraith Rotator](https://github.com/EnrikeRoe/wraith-rotator)
Herramienta CLI para Linux que automatiza la rotación de IP pública a través de ProtonVPN. Monitor TUI en tiempo real, instalador interactivo bilingüe y script autocontenido instalable via `curl` — payloads Python embebidos en base64.
`Python` `Bash` `OpenVPN` `Linux`

### [Echo-B](https://github.com/EnrikeRoe/echo-b)
Plataforma SaaS multi-tenant de chatbots con pipeline RAG propio sin frameworks. Migración de n8n + Railway a stack serverless 100% en Cloudflare — coste operativo $0/mes. API RESTful con roles, widget embebible, aislamiento total por cliente.  
`JavaScript` `Cloudflare Workers` `RAG` `LLM`

### [Omoide no Tabi](https://github.com/EnrikeRoe/omoide-no-tabi) *(privado — entrega próxima)*
App Android de diario de viaje gamificado, desarrollada en solitario. Progresión por fechas reales, misiones diarias, coleccionables y grabación de audio nativa. Arquitectura offline-first sin backend. Distribuida como APK standalone.  
`React Native` `Expo` `TypeScript` `Android`

### [TrapCraft](https://github.com/EnrikeRoe/trapcraft) *(próximamente)*
Juego de plataformas 2D desarrollado en solitario con Godot 4. Streaming de niveles por chunks, sistema de trampas por señales, coleccionables persistentes. IA de enemigos con detección por visión, persecución adaptativa, escalada de paredes y teletransporte posicional.  
`Godot 4` `GDScript`
